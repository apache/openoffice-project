## Older OpenOffice.org Releases ##

Apache OpenOffice became an incubator project on June 01, 2011. Prior to that date the OpenOffice.org project was mainly driven
by Sun Microsystems and Oracle. All installation files of the latest legacy OpenOffice.org release are available at
[Download Legacy OpenOffice.org](https://www.openoffice.org/download/legacy/index.html).
